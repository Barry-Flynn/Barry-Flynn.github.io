SQL Server版本很多，每个版本的又分为
- Enterprise（企业版，功能完备且强大）
- Standard（标准版）
- Express（精简版）等

这几种在计算能力，数据存储能力等方面都有一些差异，大家可以根据自己所需功能挑选，我作为学习使用将安装
 **Microsoft SQL Server 2012 版本的 Express（精简版）**。

本文步骤在Windows10操作系统下进行

# 安装数据库SQL Server Express
>SQL Server Express 是由Microsoft所开发的SQL Server的其中一个版本，这个版本是免费且可自由转散布（需经注册），并且可和商用程序一起使用的小型数据库管理系统。（百度百科）

- 双击执行`cn_sql_server_2012_express_sqlexpr_x64.exe`

- 点击“选项”，看一下它的安装目录
>![SQL下载2.png](https://i.loli.net/2021/10/26/RKukWFxr6gQ1fb8.png)

- 点击“安装”，选择“全新SQL Server独立安装…”
>![SQL下载3.png](https://i.loli.net/2021/10/26/5UnHWQdGElxfyS6.png)
>![SQL下载4.png](https://i.loli.net/2021/10/26/EBj3XNMDrmgsJAP.png)

- 许可条款：勾选完之后点击下一步
>![SQL下载5.png](https://i.loli.net/2021/10/26/VyGAgMqwjmSREh7.png)

- 产品更新：直接下一步
>![SQL下载6.png](https://i.loli.net/2021/10/26/n5Th2ZEis1y8Jc9.png)

- 点击安装
>![SQL下载7.png](https://i.loli.net/2021/10/26/GA9RHzqUowP6ysZ.png)

- 功能选项：全选，下一步
>![SQL下载8.png](https://i.loli.net/2021/10/26/86R3Do4cfBIQLrw.png)

- 实例配置：选择默认实例，下一步
>![SQL下载9.png](https://i.loli.net/2021/10/26/PWryc96XVH2qdQM.png)

- 服务器配置：下一步
>![SQL下载10.png](https://i.loli.net/2021/10/26/Nt56qM9KLpHxVyE.png)

- 数据库引擎配置：
1、选择混合模式

3、添加当前用户
4、下一步

- 错误报告：下一步

# 安装图形化管理界面SSMS
>SQL Server Management Studio 是用于管理SQL Server基础架构的集成环境。Management Studio提供用于配置、监视和管理SQL Server实例的工具。 此外，它还提供了用于部署、监视和升级数据层组件（如应用程序使用的数据库和数据仓库）的工具以生成查询和脚本。（百度百科）
总之，SSMS是个方便操作的管理界面（客户端工具）。
