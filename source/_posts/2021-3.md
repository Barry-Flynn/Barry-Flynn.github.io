# 操作准备
一个10多G以上大小的空U盘，一台Win10电脑（非Win10我不知是否可行）。

我的是联想ThinkPad笔记本，不同电脑操作起来可能与此文略有不同，

所以还需要准备一个善于思考能灵活变通的脑瓜儿~

# 一、Ubuntu桌面系统镜像文件下载
地址：<a href="https://cn.ubuntu.com/download/desktop"> 下载Ubuntu桌面系统 | Ubuntu </a>

我选择下载的是长期支持(LTS)版本
{% note default no-icon %} 
这篇文章下载的是Ubuntu 20.04.3 LTS，有2.9G那么大，下载时请耐心等待。
{% endnote %}

# 二、U盘引导盘制作工具(Rufus)下载
地址：<a href="https://rufus.ie/zh/"> Rufus - 轻松创建USB启动盘 </a>

我选择的是2021.08.03日更新的最新的rufus-3.15版本
{% note default no-icon %} 
类似工具其实有很多，你可以用自己熟悉的。

Rufus 是一个开源免费的快速制作 U 盘系统启动盘和格式化 USB 的实用小工具，它可以快速把 ISO 格式的系统镜像文件快速制作成可引导的 USB 启动安装盘，支持 Windows 或 Linux 启动。rufus是一款实用的自启动u盘制作工具。用户可以通过该软件快速制作linux系统或者win启动u盘，不过在这之前你需要把重要资料备份到电脑中，因为格式化后什么都找不回来了。如果身边没有系统安装盘或者刻录光盘，那么使用rufus就可以帮你安装系统了。
{% endnote %}

# 三、制作系统盘
双击运行 rufus-3.15.exe 程序，

插入你的U盘，

点击“引导类型选项”后面的选择按钮，

选择刚刚下载的ubuntu-20.04.3-desktop-amd64.iso

![rufus界面.png](https://i.loli.net/2021/09/16/2yN1L6zgoCc5MEF.png)

其他选项默认就可以，点击开始。

![Rufus弹出框.png](https://i.loli.net/2021/09/16/MvHA4wf1k82FGXm.png)

选择以 ISO镜像 模式写入，点击OK。

接下来它就会把原来的东西格式化，然后把Ubuntu烧录进去。

（注意：U盘不要留东西，资料提前备份好！格式化后就全没了！！！）

![Rufus烧录完后的U盘.png](https://i.loli.net/2021/09/16/SFqnait5Nueb48j.png)

上图就是烧录完后的U盘内文件了，

现在拔出U盘。

# 四、制作分区
按下图打开计算机管理（或者右键点击桌面“此电脑”，然后点击“管理”就行了）

![管理分区.png](https://i.loli.net/2021/09/16/xtTvZY1mygUR9w3.png)

然后点击左侧磁盘管理，选择一个你想腾出来的盘（不会影响其中文件），

我选的是E盘，右键，选择压缩卷，

![磁盘管理.png](https://i.loli.net/2021/09/16/ZF2KxUp35RIgLyl.png)

接着输入你想压缩的空间大小，

由于我笔记本盘容量小，于是就只给了50G，也就是51200MB，

你们可根据实际情况填写，当然是越大越爽，但如果不常用的话，倒也不用给太多。

![输入压缩空间大小.png](https://i.loli.net/2021/09/16/kvXctr4j3xHUygd.png)

点击压缩，放心，这不会影响磁盘原有文件的。

![磁盘压缩后.png](https://i.loli.net/2021/09/16/wxUCS7IDYbNHh4n.png)

# 五、进入BIOS界面进行设置
## 1、进入BIOS
{% note default no-icon %} 
BIOS：(基本输入输出系统)，用于电脑开机运行系统各组件时的自我检测，以及加载系统启动引导程序，BIOS设置决定系统以什么方式启动，比如硬盘启动、U盘启动等，另外BIOS控制了电脑所有的低级功能，很多系统设置需进入BIOS界面才能更改。进入BIOS的方法因电脑品牌类型不同而有所差异。
{% endnote %}

### （方法1）：通过热键进入BIOS
不同品牌的电脑进入所使用的按键不同，甚至同为联想笔记本ThinkPad但不同的系列，进入的方法都不同。你可以去网上搜一下你的电脑进入的按键是什么，比如很多电脑的方法是在启动和重新启动时按F2。

### （方法2）：通过UEFI固件设置进入BIOS
按住shift键别撒手，点击重启电脑，电脑会进行高级启动。

选择右边“疑难解答”。

![选择疑难解答.png](https://i.loli.net/2021/09/16/63dvyJtGhEY1LuN.png)

然后选择“高级选项”，

![选择高级选项.png](https://i.loli.net/2021/09/16/8H5Q2jMyLahi4qY.png)

选择“ UEFI固件设置”，

![选择UEFI固件设置.png](https://i.loli.net/2021/09/16/9ZSk1fW4gcvLy6q.png)

现在将刚刚那个系统U盘插入电脑，

点击“重启”，通过UEFI固件设置进入BIOS。

![选择重启以更改UEFI固件设置.png](https://i.loli.net/2021/09/16/UqjR6bwXEaCQZuI.png)

## 2、对BIOS进行U盘启动设置
（以下内容，不同电脑操作键可能有差别，不同的主板BIOS的界面也是不同的）

### （1）设置Secure Boot
通过方向键，往右选择Security，再往下选择Security Boot，回车。

![选择Security.png](https://i.loli.net/2021/09/17/qbZ8FyNjBWfQPxa.png)

选中SecureBoot并回车，上下键更改为Disabled，回车选中。

这里的作用是禁用UEFI安全启动（Disable Secure Boot for ASUS Motherboard），

因为如果主板设置了uefi安全启动，会与ubuntu安装的的第三方驱动发生冲突。

![更改SecureBoot值.png](https://i.loli.net/2021/09/17/R6lr4bmBzTwYCoq.png)

按Esc退出Security。

### （2）将USB HDD置顶
通过方向键，往右选择Starup，再往下选择Boot，回车。

![选择Starup.png](https://i.loli.net/2021/09/17/F687eEnKyoS4Db5.png)

通过上下键选到USB HDD，

它后面的单词是我这次使用的U盘的牌子：金士顿（Kingston）

![选择Ubuntu系统.png](https://i.loli.net/2021/09/17/rbDkoWGfv6ijN5K.png)

通过按 “+”（加号键）把Ubuntu系统顺序设为首位，（其他笔记本请参考你们的按键）

这里的作用是更改Ubuntu与Windows的启动顺序。

![把Ubuntu顺序置顶.png](https://i.loli.net/2021/09/17/zS13kqeAhCPXVdf.png)

按 “F10” 保存退出。

# 六、安装Ubuntu
上一个操作完成后出现四行黑屏代码，

通过上下键选择第二行的Install Ubuntu并回车，进行安装Ubuntu。

![U盘安装界面.png](https://i.loli.net/2021/09/17/E9dw6Th4yrYAJgG.png)

启动中，静静等待。

![Ubuntu系统启动中.png](https://i.loli.net/2021/09/17/wdQVUAak7cN26bP.png)

## 1、欢迎
左边语言选择中文简体，然后选择右边安装Ubuntu，

![安装Ubuntu.png](https://i.loli.net/2021/09/17/DZ9z4NxmIoKsBGu.png)

## 2、键盘布局
默认中文就行，继续，

![安装Ubuntu-2-键盘布局.png](https://i.loli.net/2021/09/17/89UcG6fWsBFA1oT.png)

## 3、无线网络
可以先跳过不连接，你选择现在连上也当然可以，下面继续。

## 4、更新和其他软件
我选择最小安装（这个选项也是速度最快的），不安装多余应用，以后需要用到了再说。

当然你可以选择正常安装，会给你安装一些办公软件、游戏和媒体播放器啥的，不过时间会很长。

![安装Ubuntu-4-更新和其他软件.png](https://i.loli.net/2021/09/17/25A8p4Vh71lXgoD.png)

## 5、安装类型
第一次安装的朋友在这一步请注意！！！一定要选“其他安装”！！！另一个选项会清除整个磁盘的所有文件，也就是你的Windows系统会被刷机！！！手别抖！！！一定要选“其他安装”！！！

![安装Ubuntu-5-安装类型.png](https://i.loli.net/2021/09/17/zTCJ94eYHZNAgpM.png)

{% note default no-icon %} 
接下来我们对Ubuntu进行分区，在分区之前先介绍一下Linux的文件系统：

<b>/boot</b>:存放linux内核，用来引导系统的。如果是Legacy启动就要设置引导，UEFI就不用设置这个（UEFI要设置EFI引导区）

<b>swap</b>：交换区，用作虚拟内存，这个一般和自己的物理内存一般大

<b>/</b>：根目录，主要用来存放Linux系统文件

<b>/usr</b>:存放用户程序，一般在/usr/bin中存放发行版提供的程序，用户自行安装的程序默认安装到/usr/local/bin中

<b>/home</b>:存放用户文件
{% endnote %}

滚动鼠标滑轮，找到与你刚刚在Windows系统里分配给Ubuntu的相近大小的那个“空闲”磁盘空间（freespace），就像我刚刚给Linux分了50G大小，我就要选择下图那个50多个G（53688MB）的那个空闲空间。

![安装Ubuntu-5-安装类型2.png](https://i.loli.net/2021/09/17/eWngyC5Zk7p12YR.png)

### （1）分配swap交换区
选中那个空闲空间，点击左下角加号“+”，会弹出一个弹窗，

听朋友说，如果运行内存足够，其实swap交换区不设置也行，

我的电脑运行内存有8G的物理内存，应该够用，所以这个swap虚拟内存我就不设置很大了

![电脑物理内存RAM.png](https://i.loli.net/2021/09/17/1XVhQPyb6ogli5D.png)

大小我给了1G，即“<b>1024MB</b>”，

分区选择第二个“<b>逻辑分区</b>”，位置选择“<b>空间起始位置</b>”，

用于“<b>交换空间</b>”，挂载点填“ / ”，也就是用来存放Linux系统文件的根目录，

![分配swap分区.png](https://i.loli.net/2021/09/17/qdXYcKBo4h8fARV.png)

然后点击OK。

### （2）分配EFI引导区
选中那个空闲空间，点击左下角加号“+”，

大小我给了“<b>100MB</b>”，

分区选择“<b>逻辑分区</b>”，位置选择“<b>空间起始位置</b>”，

用于“<b>EFI系统分区</b>”模式，

![分配EFI引导.png](https://i.loli.net/2021/09/17/7lJksmynocCp6OA.png)

然后点击OK。

### （3）分配 / 分区（根目录）
选中那个空闲空间，点击左下角加号“+”，

相当于Windows的C盘，

大小我给了10 G，即“<b>10240MB</b>”，

分区选择第一个“<b>主分区</b>”，位置选择“<b>空间起始位置</b>”，

用于“<b>Ext4日志文件</b>”，挂载点填“<b>/</b> ”，

![分配根目录.png](https://i.loli.net/2021/09/17/HoxtLwAl6JIyVZF.png)

然后点击OK。

### （4）分配home分区
选中那个空闲空间，点击左下角加号“+”，

相当于Windows的C盘下的“我的文档”，

大小：如果你不分配其他分区的话，那个空闲空间的剩余大小全给它就行，

分区选择“逻辑分区”，位置选择“空间起始位置”，

用于“Ext4日志文件”，挂载点填“ /home ”，

![分配home分区.png](https://i.loli.net/2021/09/17/pKl2DcEh5NU8Fxi.png)

 然后点击OK。

### （5）分配一些其他区
我只是初学，暂时用不到那么多的分区，所以把50多G空间都分给了上面那几个分区。想分其他分区的朋友们，感兴趣可以自行了解。

### （6）选择“安装启动引导器的设备”
下面那个“安装启动引导器的设备”选择我们刚刚分配的EFI引导区，

注意不要选成Windows的efi了！

![Windows的EFI.png](https://i.loli.net/2021/09/17/lJ7sgCqBkcRHmwD.png)

而是要选刚刚咱们分配给Ubuntu的efi，比如我刚刚分配的100M的那个，

![Ubuntu的EFI.png](https://i.loli.net/2021/09/17/XTROxJWrIV2Zitk.png)

## 6、立即安装
选择现在安装，

确认一下信息，无误后继续写入磁盘，

![安装Ubuntu-5-写入磁盘.png](https://i.loli.net/2021/09/17/Ld8GxMP6bqAW1pI.png)

时区选默认的上海就可以，点击继续，

![安装Ubuntu-6-时区.png](https://i.loli.net/2021/09/17/9mndRu2AZiJeDow.png)

下面的三个名字全输一样的就行，也别太长，

密码建议000000，或者123456等容易记忆又方便的，因为以后会经常输入。

选择“登录时需要密码”，点击继续。

![安装Ubuntu-7-名称与密码.png](https://i.loli.net/2021/09/17/h2OetW1bGvC5EBy.png)

下面开始安装，漫长的等待中你可以看看轮播图了解一下Ubuntu。

![安装Ubuntu-安装中.png](https://i.loli.net/2021/09/17/6nmAjvL4c1xY8zD.png)

然后Ubuntu安装成功，现在重启。

![Ubuntu安装成功-等待重启.png](https://i.loli.net/2021/09/17/M52mDN3kLBOEwjd.png)

这个页面提示我们拔出U盘，并按下回车。

![Ubuntu提醒拔出U盘并回车.png](https://i.loli.net/2021/09/17/lVNZsPtizORXuSr.png)

下面进入了Ubuntu的grub引导界面，以后电脑每次开机时都会出现，

可以通过上下键选择进入Ubuntu系统还是Windows系统，如下图：

选择第一行按回车就是进入Ubuntu，

选择第三行按回车就是进入Windows，

第二行和第四行不用管，

长时间不选择，默认进入第一个。

![双系统的进入选择界面.png](https://i.loli.net/2021/09/17/i4xwUGeLNk75tOY.png)

安装完成后第一次打开时会有几个窗口，除了“Help Improve Ubuntu”那个窗口选择第二个选项`No`（不发送系统信息）之外，其他全跳过就行了。

这样我们就能使用Ubuntu系统了。

本文仅供参考，我是初学Linux，电脑磁盘容量不是很多，所以只分了50 G，
我所观看的教程中大多都是分配上百G，请根据你的情况进行合理调整分区大小。
若有疏漏之处，欢迎评论或联系我指出。
