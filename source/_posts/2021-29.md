# 基本语句

## printf 输出语句

打印文字（其中，`\n`是换行符，`\t`是空格）

```C
#include<stdio.h>
int main()
{
    printf("我是百里飞洋\n");
    printf("星河滚烫\t");
    printf("无问西东\t");
    printf("Hello World!");
    printf("%d\n",23+43);
    printf("%d\n",100-23);
    return 0;
}
```
效果：

    我是百里飞洋
    星河滚烫 无问西东
    Hello World!
    66
    77

打印基本类型
```C
#include<stdio.h>
int main()
{
    int height=18;
    float price=11.1;
    double pi=3.1415926;
    char sex='M';

    printf("%d,%f,%f,%c\n",height,price,pi,sex);
    printf("%s",我是百里飞洋);
    return 0;
}
```
效果：

    18,11.1,3.1415926,M
    我是百里飞洋

**`%m.nf`前面的小数部分规定输出结果所占空格数**


m表示输出数占的总空格数

n表示小数点后保留的位数

- 若m.n为正，位数不够，输出数据前面补空格
- 若m.n为负，位数不够，输出数据后面补空格



```C
#include<stdio.h>
int main()
{
    double a,b,c;
    a=3.67;
    b=5.43;
    c=6.21;
    printf("设定的a是%5.2f",b是%-5.2f,c是%7.4f\n,a,b,c);
    return 0;
}
```
输出结果：

    设定的a是 3.67",b是%5.34 ,c是 6.2100


**`%e`规定以科学计数法输出结果**

```C
#include<stdio.h>
int main()
{
    print("傻狍子的身高是%10.2e厘米\n,123.456");

    return 0;
}
```
输出结果：

    傻狍子的身高是 1.23e+002厘米
也就是1.23*10^2厘米

## getchar()和putchar()语句

`getchar()`从键盘得到某个字符

`putchar()`呈现某个字符

```C
//此程序可以将3个大写字母转换为小写字母
#include<stdio.h>
int main()
{
    char a,b,c;
    a=getchar();
    b=getchar();
    c=getchar();
    a=a+32;
    b=b+32;
    c=c+32;
    putchar(a);
    putchar(b);
    putchar(c);

    return 0;
}
```

## 数学运算符
除了`+`加、`-`减、`*`乘、`/`除、`%`取余之外，更高级的数学运算需要添加如下预处理命令

| 5 | `<` `<=` `>=` `>` |

| 8 | `丨丨`两边满足一个 |
