>本文为让<code>深色模式</code>下的博客网页
增加<code>星空背景和流星特效</code>配置的小教程，
该效果如下：

![深色模式下星空背景和流星特效](https://cdn.jsdelivr.net/gh/Justlovesmile/CDN2/post/202108121834269.gif)

>本文以Hexo博客的Butterfly主题为例，其他博客框架、主题、网页如何生效请参考本文自行尝试，我不做解答。

# 1. 插入Canvas标签

首先打开Butterfly主题的<code>_config.yml</code>文件或者使用HTML直接插入，找到配置文件对应的<code>inject</code>部分，插入`<canvas id="universe"></canvas>`
![插入Canvas标签](https://cdn.jsdelivr.net/gh/Justlovesmile/CDN2/post/202108121759902.png)

代码的这一部分要求`data-theme`也就是主题，必须为dark暗色主题（即夜间模式），因此仅在暗色主题生效。随后将js文件添加到配置文件的`inject`处或者其他需要的位置。

# 3. CSS样式

将下方代码添加到Butterfly主题配置文件的`inject`的`head`里
作用是将此加入到每个页面的头部代码中

```css
/* 背景宇宙星光  */
<style type="text/css">
      #universe { 
        display: block;
        position: fixed;
        margin: 0;
        padding: 0;
        border: 0;
        outline: 0;
        left: 0;
        top: 0;
        width: 100%;
        height: 100%;
        pointer-events: none;
        z-index: 1;
        }
</style>
```

流星特效至此已经生效，
保存代码，预览本地，部署页面，备份源码。


星空背景自己在网络上找一张就行了，或者干脆不用星空背景，其他普通图片作为背景，用此特效也好看。





>这个教程学习自[Justlovesmile's BLOG](https://blog.justlovesmile.top/posts/6a260bf6.html)，在原文基础上有改动和补充。
该特效有一个小Bug是`暗色模式`调成`亮色模式`后流星特效还在，只是静止不动使其不明显了，没有清空画布。
